# Retail Operations Analytics - Excel-Based Performance Monitoring

This project presents a multi-page **Excel dashboard** designed to track **retail performance metrics** at a regional level. It was built using **Power Query**, advanced Excel formulas, and interactive **Pivot Tables** to enable dynamic KPI reporting and real-time variance tracking.

---

## 📌 Project Summary

- Developed for internal reporting during a retail analytics internship.
- Tracks key metrics including **Revenue, Profit, Target Comparison, Refund Rate, Customer Segments, Store Performance**, and more.
- Dashboard supports **quarterly/monthly views**, **store-level filtering**, and **variance analysis**.
